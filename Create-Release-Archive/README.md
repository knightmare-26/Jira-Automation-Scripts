# Jira Version Manager

A web-based tool to manage Jira fix versions across multiple projects.

## 🚀 Features
- **Project Tracking:** Add/remove projects from your Jira account in bulk.
- **Active projects:** Select and save sets of projects to work with.
- **Version Management:** Batch create, release, archive, and **rename** versions across multiple projects.
- **Shortcuts:** Save custom project and version configurations as quick-access shortcuts.
- **Privacy First:** Each user has their own private Jira credentials and workspace, securely isolated in the cloud.

## ⚙️ Setup Instructions
1. **Connect to Jira:** 
   - Navigate to the **⚙️ Config** page (the app defaults here if not configured).
   - Enter your Jira Base URL, Email, and API Token.
   - Click "Save Configuration". Your credentials are saved privately to your user profile.
   
2. **Personalize Your Workspace:**
   - Go to **📂 Manage Projects**.
   - Use the **"⚙️ Manage Tracked Projects"** tab to add or remove the projects you want to work with using the batch search bars.
   - In the **"🎯 Active projects"** tab, check the boxes for the projects you want to manage currently.

3. **Performing Actions (within 📂 Manage Projects):**
   - Use the **"🚀 Create Versions"** tab to define and create new versions across your selected projects.
   - Use the **"📦 Release/Archive/Rename"** tab to batch update the status or rename your fix versions across all active projects.
