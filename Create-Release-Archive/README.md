# Jira Version Manager

A web-based tool to manage Jira fix versions across multiple projects.

## 🚀 Features
- **Project Tracking:** Add/remove projects from your Jira account.
- **Active Workspace:** Select and save sets of projects to work with.
- **Version Management:** Batch create, release, and archive versions.
- **Shortcuts:** Save custom project and version configurations.

## ⚙️ Setup Instructions
1. **Connect to Jira:** 
   - Navigate to the **⚙️ Config** page.
   - Enter your Jira Base URL, Email, and API Token.
   - Click "Save Configuration".
   
2. **Personalize Your Workspace:**
   - Go to **📂 Manage Projects**.
   - Use the **"Manage Tracked Projects"** tab to add the projects you want to work with.
   - In the **"Active Workspace"** tab, check the boxes for the projects you want to manage currently.

3. **Performing Actions:**
   - Use **🚀 Create Versions** to define and create new versions across your selected projects.
   - Use **📦 Release/Archive** to batch update the status of your fix versions.
